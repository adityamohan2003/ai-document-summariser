# 🤖 AI Document Summariser Tool

## 🌐 Live Demo
👉 [Try it here](https://ai-document-summariser-w8m6.onrender.com/)

## Overview
A Python tool that uses the Groq AI API to instantly analyse and summarise any text. 
Paste in any article, report, or document and get back a clean summary, key points, and main topics.

## Features
- Concise 3-4 sentence summary
- Five extracted key points
- Main topics identified
- Works with any text — articles, reports, research papers

## Example Output
Input: A news article about AI in healthcare

Output:
- Clean summary of the article
- Five key points extracted
- Main topics covered

## Tools & Libraries
- Python
- Groq API (llama-3.3-70b-versatile model)
- Command line interface

## How to Run
1. Clone this repository
2. Install dependencies: `pip install groq`
3. Get a free API key at console.groq.com
4. Add your API key to the code
5. Run: `py summariser.py`
